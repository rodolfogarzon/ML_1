Overview

The purpose of this analysis is to use data from accelerometers on a bell, forearm, arm and dumbbell of six participant to predict how well the participant do barbell lifts in 5 different ways. the outcome will be the variable classe, class A corresponds to the specified execution of the exercise, while the other 4 classes correspond to common mistakes. In order to analyze the correlation between the different groups of variables, we need to install corrplot package
